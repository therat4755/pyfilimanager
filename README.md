# File Manager on tkinter

![Made by therat4755](splash.png)

## Overview
A simple file management system that allows users to perform basic file operations.

## Features
- Create, read, update and delete files
- Navigate through directories
- Copy and move files
- Search for files and directories
- View file properties and metadata
- Sort files by name, date, size, or type

## Technical Details
### Core Components
- File Operations Module
- Directory Navigator
- Search Engine
- User Interface
- Permission Handler

### Supported File Operations
- Create new files/folders
- Delete files/folders
- Rename files/folders
- Copy/Move files
- Change file permissions
- View file details

## System Requirements
- Operating System: Windows/Linux/MacOS
- Minimum RAM: 2GB
- Storage: 500MB free space
